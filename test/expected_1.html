Introduction

This is a small test of how to create a new page and section. If you are a developer, then you should know that the entire application code is inside the global namespace window.app

Cache module

Module to handle different kind of caches in the client-side. All the logic is wrapped in the `app.cache` namespace.

Static methods

get(url, params)

Type Name Description
String url The relative url
JSON params JSON object to configure the request behaviour
Function params.success Callback to execute with the successful response from the server
Function params.fail Callback to execute when the request fails

It sends a GET request to the the server via AJAX

example

app.xhr.get('/users.json', { success: function(res) { console.log(res); }, fail: function() { console.log('Error!') } })
